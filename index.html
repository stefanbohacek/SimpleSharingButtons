Download the icons

You can download Font Awesome from their official website. Alternatively you can include this code inside your head element:
<link href="https://use.fontawesome.com/releases/v5.0.9/css/all.css" rel="stylesheet">
Make sure you have the latest version of Font Awesome; alternatively, get your embed code here. Note that you will also need to apply custom CSS (for color, hover style etc).

About this site

Simple Sharing Buttons Generator is an open source tool made by @fourtonfish that lets your create light-weight sharing buttons for social networks, including Twitter, Facebook, or Google+, that speed up your website by avoiding downloading unnecessary JavaScript files and they keep your user's activity private.

You can choose from several styles, including SVG images and Font Awesome. This is site is free to use, check out above for donation options.

Some of the sites using Simple Sharing Buttons Generator:
